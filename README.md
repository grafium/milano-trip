# Milánó 2026 — Családi hétvége · Trip Planner

Interaktív, TravelPerk-stílusú útiterv-oldal a 2026. aug 28–30. milánói családi hétvégéhez.

- 4 kattintható fül: Áttekintés + 3 napi terv
- Leaflet térkép napi útvonalakkal (CARTO/OSM csempék)
- Malpensa Express transzfer, ATM jegyinfók, költségösszesítő

## Publikálás GitHub Pages-en

1. Repo: `milano-trip` (vagy tetszőleges név)
